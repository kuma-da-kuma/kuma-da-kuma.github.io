HTML & CSS の開発環境を用意しよう！

自分のPCでWebページを作成する方法を学習しましょう
  • Atomをダウンロードして使おう
  • デベロッパーツールを使ってみよう
  • Git hubへ登録してネット上にページを作ろう

Atomをダウンロードして使おう

ネット上で　Atom　を検索してダウンロードします。必要があれば日本語で表示出来るように設定してみましょう。

デベロッパーツールを使ってみよう

　Google Chromeの設定の中の その他のツールにある デベロッパーツールを使用して、様々なページの構成を確認してみましょう。

Git hubへ登録してネット上にページを作ろう

ネット上で　Git hub　を検索して無料登録します。リポジトリを新たに作成したら緑色のボタンを押して、READMEをクリックし、再度緑色のボタンをクリックします。Create new file から index.htmlとstyleseet.cssを作成し、コードを準備していれば貼り付けてみましょう。
